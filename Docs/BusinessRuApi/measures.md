# Business.ru API: Единицы измерения (measures)
**Описание модели:** Cправочник единиц измерения  
**URL ресурса:** https://myaccount.business.ru/api/rest/measures.json  
**Разрешенные запросы:** get(чтение), post(создание), put(изменение), delete(удаление)  

| Параметр | Тип      | Обязателен в запросах | Внешний ключ к модели | Значение по умолчанию | Описание |
| --- |----------| --- | --- | --- | --- |
| id  | long     | put, delete |     |     | Идентификатор |
| name | string   | post |     |     | Наименование |
| short_name | string   |     |     |     | Краткое наименование |
| default | bool     |     |     |     | Флаг (0-обычная единица измерения, 1-Единица измерения, выставляемая по умолчанию при создании нового товара) |
| okei | long     |     |     |     | Код единицы измерения из Общероссийского классификатора единиц измерения |
| archive | bool     |     |     |     | Флаг (0-элемент не перемещен в архив, 1-элемент перемещен в архив) |
| updated | datetime |     |     |     | Время последнего обновления |
| deleted | bool     |     |     |     | Строка удалена (перемещена в корзину) |
