# Business.ru API: Значения списковых дополнительных полей (additionfieldslists)

**Описание модели:** Список значений дополнительных полей типа "список" и "переключатель"  
**URL ресурса:** https://myaccount.business.ru/api/rest/additionfieldslists.json  
**Разрешенные запросы:** get(чтение), post(создание), put(изменение), delete(удаление)

Модель additionfieldslists отображает значения всех дополнительных полей типа "список" и "переключатель". Модель позволяет добавлять новые значения в уже существующие дополнительные поля. Модель позволяет удалять и редактировать значения дополнительных полей. Получить список всех доступных дополнительных полей можно с помощью GET-запроса с параметром **"help=1"**.

| Параметр | Тип | Обязателен в запросах | Внешний ключ к модели | Значение по умолчанию | Описание |
| --- | --- | --- | --- | --- | --- |
| id  | int | delete, put |     |     | Идентификатор |
| additionfields_id | int | post |     |     | Ссылка на идентификатор дополнительного поля типа "список" и "переключатель". Идентификатор можно получить, указав в GET-запросе параметр help=1 |
| field_value | string | post, put |     |     | Название значения дополнительного поля |
