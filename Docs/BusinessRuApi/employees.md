# Business.ru API: Сотрудники

## Список сотрудников (employees)

**Описание модели:** Cписок сотрудников организаций аккаунта

**URL ресурса:** https://myaccount.business.ru/api/rest/employees.json

**Разрешенные запросы:** get(чтение), post(создание), put(изменение), delete(удаление)

**Специальные параметры модели:**

**with_access** - при передаче в GET запросе with_access = 1, информация о каждом из сотрудников будет дополнена данными об объектах системы, к которым данный сотрудник имеет доступ, включая:

*   **retailpoints_ids** - массив идентификаторов розничных точек, в которые сотрудник может входить как кассир;
*   **organizations** - секция **ids** - массив идентификаторов организаций, к которым принадлежит сотрудник, секция **default** - организация по умолчанию;
*   **stores** - секция **ids** - массив идентификаторов складов, к которым сотрудник имеет доступ, секция **default** - склад по умолчанию;
