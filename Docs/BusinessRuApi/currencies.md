# Business.ru API: Валюты (currencies)
**Описание модели:** Cправочник валют  
**URL ресурса:** https://myaccount.business.ru/api/rest/currencies.json  
**Разрешенные запросы:** get(чтение)

| Параметр | Тип | Обязателен в запросах | Внешний ключ к модели | Значение по умолчанию | Описание |
| --- | --- | --- | --- | --- | --- |
| id  | long | put, delete |     |     | Идентификатор |
| name | string |     |     |     | Наименование валюты |
| short_name | string |     |     |     | Сокращенное наименование валюты |
| name_iso | string |     |     |     | Буквенный код валюты по ISO 4217 |
| code_iso | long |     |     |     | Цифровой код валюты по ISO 4217 |
| symbol_iso | long |     |     |     | Символ валюты по ISO 4217 |
| okv | string |     |     |     | Код ОКВ |
| default | bool |     |     |     | Флаг ( 0-обычная валюта, 1-валюта принятая в системе по умолчанию ) |
| user | bool |     |     |     | Флаг ( 0-при расчетах используется курс ЦБ, 1-при расчетах используется пользовательское значение курса ) |
| user_value | float |     |     |     | Пользовательское значение курса валюты |
| name1 | string |     |     |     | Первая множественная форма |
| name2 | string |     |     |     | Вторая множественная форма |
| name3 | string |     |     |     | Третья множественная форма |
