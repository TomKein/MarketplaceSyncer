# Business.ru API: Страны (countries)
Страны (countries)  
**Описание модели:** Cправочник стран  
**URL ресурса:** https://myaccount.business.ru/api/rest/countries.json  
**Разрешенные запросы:** get(чтение)

| Параметр | Тип    | Обязателен в запросах | Внешний ключ к модели | Значение по умолчанию | Описание |
| --- |--------| --- | --- | --- | --- |
| id  | long   |     |     |     | Идентификатор |
| name | string |     |     |     | Наименование |
| full_name | string |     |     |     | Полное наименование |
| international_name | string |     |     |     | Международное наименование |
| code | string |     |     |     | Цифровой код страны |
| alfa2 | string |     |     |     | Двухсимвольное обозначение страны |
| alfa3 | string |     |     |     | Трехсимвольное обозначение страны |
