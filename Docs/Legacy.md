# Статус Legacy (Проект Selen)

## Общее состояние
Проект `Selen` (расположен в папке `Legacy/Selen`) представляет собой устаревшее (legacy) приложение.
Качество кода оценивается как низкое, соответствующее старым стандартам .NET Framework 4.7.1.

## Цели и задачи
- **Статус**: Deprecated (Устаревший). Активная разработка и поддержка не ведутся.
- **Назначение**: Используется **исключительно как референс** (справочный материал) для понимания логики работы с площадками, механик и бизнес-правил.
- **Рефакторинг**: Допускается минимальный рефакторинг с целью улучшения читаемости и сокращения объема кода, чтобы упростить извлечение знаний.
- **Миграция**: Новая функциональность реализуется с нуля в проекте `MarketplaceSyncer.Service` на .NET 10. Прямой перенос кода (copy-paste) не рекомендуется без глубокого анализа и приведения к современным стандартам.

## Известные проблемы
- Устаревший стек технологий.
- "Катастрофическое" качество кода (по словам владельца).
- Возможные проблемы с надежностью и стабильностью.

## Структура
Проект вынесен в отдельную директорию `Legacy/Selen` для очистки корня решения и явного разделения между старым и новым кодом.
В solution файл (`selen.sln`) проект добавлен по новому пути.
